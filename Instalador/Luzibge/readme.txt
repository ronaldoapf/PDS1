Instruções:

Para que o Software funcione corretamente, é necessário ter o Python e a biblioteca pandas instalado em seu computador.

Para que isso seja instalado automaticamente, é necessário que adicione o Python ao PATH na hora da instalação do Python.

Preencha o checkbox onde diz: "Add Python X.x to PATH" quando abrir o instalador do python automaticamente.

Caso não seja feita a instalação correta das dependências (Python e pandas), na página de tutorial existe um passo a passo mostrando o que fazer para que o programa funcione corretamente.

1. Como a aplicação funciona

Programa feito consiste em um aplicação na qual o usuário deve selecionar planilhas do IBGE, codificadas, para decodificá-las e salvar uma nova planilha apenas com as colunas que lhe interessa.

O processo de decodificação é realizado através de uma planilha de índices que deve ser feita manualmente para cada grupo de planilhas codificadas.

Dependências:

  • Baixar planilhas do IBGE: As planilhas podem ser baixadas diretamente pelo site do IBGE.
    Obs: Após realizar o download das planilhas, sugerimos que não altere o nome das mesmas, para facilitar a geração da planilha de índices, assim como é explicado na seção de Como gerar planilha de índices

  • Formato das planilhas: Todas as planilhas utilizadas nesta aplicação devem estar no formato .csv, que é um formato que separa os valores utilizando vírgula e informa as colunas na primeira linha do arquivo. Portanto, ao baixar as planilhas do IBGE, elas provavelmente virão no formalo .xlsx, e devem ser convertidas para .csv (pode-se utilizar ferramentas gratuitas como o Libre Office Calc para realizar tal conversão)

  • Python: É necessário ter instalado, em seu computador, alguma versão do python 3.7 ou superior

2. Como utilizar a aplicação

Passo 1: Clicar no botão Iniciar.

Passo 2: Selecionar as planilhas que foram baixadas pelo IBGE, que possuem colunas como “Var 01, Var02”, (todas as planilhas deverão estar no formato .csv).

  • Escolha as planilhas codificadas.
  • Selecione a planilha de índice correspondente.
  • Clique no botão "próximo" .

Passo 3: Clique no nome de uma das planilhas listadas, para carregar suas colunas.

  • (Opcional) Digite termos como "Var", "media", dentre outros, para filtrar entre as colunas codificadas e decodificadas.
  • Para selecionar uma coluna como interessante, basta clicar no "ícone vermelho" correspondente.
  • Para renomear uma coluna primeiro ela deve ser marcada como interessante, conforme passo anterior. Após selecionada, basta clicar no campo "valor renomeado" correspondente e digitar o novo nome.

Passo 4: Salvar nova planilha.

  • Após selecionadas as colunas de interesse, clique no botão "Salvar" (Botão com ícone de disquete).
  • Informe o nome da nova planilha e, por último, clique novamente no botão "Salvar".
  • Obs: A nova planilha será salva no mesmo diretório da planilha original.

3. Como gerar planilha de índices

Passo 1 : Baixar as planilhas - Disponibilizadas no site do IBGE


Passo 2: Estrutura do arquivo .csv

  • Primeira linha do arquivo csv deve ser: "chave,valor", pois representa as colunas;

  • A estrutura do arquivo deverá seguir o seguinte padrão "nome da planilha"_"nome coluna decodificada","nome da coluna decodificado". Exemplo: " basico_var01, responsaveis pelo domicilio ";



Passo 3: Mapear índices para as planilhas, que podem ser de dois tipos:

  • Multivaloradas: Podem assumir diferentes valores decodificados “1”, “2”,”3”,...;
  • Simples: Assume um único valor decodificado;
